Key slash commands

  • /model — switch the active model mid-session
  • /mcp — list or manage MCP server connections
  • /usage — check your premium request consumption
  • /resume — pick up the last session where you left off
  • /clear — wipe current session context (pivot to a new task)
  • /context — show what files/URLs are currently attached
The full slash-command list lives in docs.github.com/en/copilot/reference/copilot-cli-reference. /clear is especially useful when you want to pivot to an unrelated task without starting a new process. copilot --continue (from the shell, not inside the session) resumes the last session. Auto-compaction kicks in at 95% context with no user action required.

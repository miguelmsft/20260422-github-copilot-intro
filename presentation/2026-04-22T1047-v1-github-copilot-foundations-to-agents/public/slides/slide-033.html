Interactive essentials

  • Shift+Tab — cycle modes (ask → plan → autopilot)
  • !cmd — run shell directly (no model call)
  • @path/to/file — attach file as context (Tab-completes)
  • Enter sends a prompt · Esc stops a running operation
Live demo these. The ! and @ prefixes are what make the CLI feel native to a terminal power user. Additional keys worth mentioning verbally but not putting on the slide: Enter sends a prompt; Esc stops a running operation; copilot --continue resumes the last session; auto-compaction kicks in at 95% context with no user action required. Full slash-command list lives in docs.github.com/en/copilot/reference/copilot-cli-reference.

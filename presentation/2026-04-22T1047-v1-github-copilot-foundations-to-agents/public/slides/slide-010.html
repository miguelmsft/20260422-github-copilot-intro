Common agent patterns

ReAct
Interleave reasoning and action, one step at a time.
Copilot's default Agent is ReAct-ish.
Plan-and-Execute
Plan everything first, then execute; re-plan if reality drifts.
Copilot's Plan mode.
Reflexion
Verbal self-critique stored between trials.
Shows up when agents loop on test failures.
Orchestrator-Workers
One agent delegates to specialized sub-agents.
Custom subagents. CLI /fleet command.
Map to Copilot: the default Agent is ReAct-ish. Plan mode is Plan-and-Execute. Custom subagents and the CLI /fleet command are Orchestrator-Workers. Reflexion shows up when agents loop on test failures.

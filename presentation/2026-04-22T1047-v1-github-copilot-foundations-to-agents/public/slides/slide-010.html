Common agent patterns

┌─────────────────────────┬─────────────────────────┐ │ **ReAct** │ **Plan-and-Execute** │ │ Interleave reasoning │ Plan everything first, │ │ and action, one step │ then execute; re-plan │ │ at a time │ if reality drifts │ ├─────────────────────────┼─────────────────────────┤ │ **Reflexion** │ **Orchestrator-Workers**│ │ Verbal self-critique │ One agent delegates to │ │ stored between trials │ specialized sub-agents │ └─────────────────────────┴─────────────────────────┘
Map to Copilot: the default Agent is ReAct-ish. Plan mode is Plan-and-Execute. The CLI /fleet command is Orchestrator-Workers. Reflexion shows up when agents loop on test failures.

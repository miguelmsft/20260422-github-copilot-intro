Instructions vs Prompt files

Custom instructionsPrompt files (*.prompt.md)
When usedAutomatic, always-onManual, on demand
How invokedEvery chat turn/review-api, /scaffold-form, …
Scope"Rules about this repo/you/org""This specific reusable task"
SurfaceAll Copilot surfacesIDEs only (VS Code, VS, JetBrains)
Rule of thumb: instructions = nouns ("this is Vue 3"), prompt files = verbs ("review this API for security issues").

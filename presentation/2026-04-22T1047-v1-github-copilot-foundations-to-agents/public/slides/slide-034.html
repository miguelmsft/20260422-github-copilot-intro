Key slash commands

Modes & models

  • /modelPick a model (Sonnet, GPT-5.x, Opus, auto, …)
  • /planEnter plan mode
  • /askOne-off question (no history)
  • /agentPick a custom agent for the next turn

Context & sessions

  • /contextVisual breakdown of context usage
  • /compactManually compress history
  • /resumePick a past session to resume
  • /add-dirAdd another trusted directory

Safety & review

  • /diffShow pending file changes
  • /undoRewind to an earlier checkpoint
  • /mcpList / add MCP servers
  • /envShow loaded instructions, skills, plugins

Account & help

  • /usagePremium requests, tokens, session length
  • /login · /logoutOAuth sign-in / sign-out
  • /shareExport conversation (markdown / HTML)
  • /help or ?Full command list
Don't read every row — point at the four columns and highlight one from each. Most-used in practice: /model, /context, /diff, /usage. /resume is the hidden gem — preserves full history from a prior session. Full reference: docs.github.com/en/copilot/reference/copilot-cli-reference
